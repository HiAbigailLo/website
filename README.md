# abigaillo.com

Personal portfolio site built with plain HTML, CSS, and JavaScript. Hosted on Firebase.

## Files

- `index.html` — all site content and structure
- `styles.css` — all styling and layout
- `script.js` — scroll animations, nav behavior, mobile menu
- `firebase.json` — Firebase Hosting config

## Commands

Preview locally:
```bash
python3 -m http.server 3000
```

Deploy:
```bash
firebase deploy
```
